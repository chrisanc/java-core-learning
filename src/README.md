## Java core concepts (my learning path)
I made this repository to share my Java knowledge with everyone and to make you notice how much of Java I know and understand. My goal is to keep learning and adapting to the language, as it's my favorite one for many reasons:

Pure OOP implementations: I'm a OOP because I feel the paradigm is very comfortable to work with, as well as very readable for other developers.
Strict Typing: I love the fact that you got to explicitly define the type of the variables so you have a better control. There's also the generic types and wildcards that are very powerful.
Enterprise-level Software: Java is the preferred language for enterprise-level software, so if you want to be a great SWE it's the best option out there.
The goal of this repository is to contain all my Java learning path and current knowledge.

From LeetCode solutions to pure Java implementations, this is all you need.
